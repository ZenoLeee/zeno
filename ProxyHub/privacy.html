ProxyHub 隐私政策

最后更新日期：2025年2月3日

我们承诺保护您的隐私。本扩展程序的所有数据都存储在您的本地设备中，我们不会收集、存储或传输任何个人数据到我们的服务器。

1. 我们收集的信息

1.1 本地存储的数据

ProxyHub 使用 Chrome Storage API 在您的本地设备上存储以下信息：

  • 订阅链接：您添加的 Clash 订阅 URL（包括可能的认证令牌）
  • API 配置：Clash API 地址和密钥
  • 显示设置：您的界面偏好设置（如深色模式、双列显示等）
  • 节点收藏：您收藏的节点列表
  • 流量信息：从订阅 URL 获取的流量使用情况（仅存储在本地）

1.2 云同步数据（可选）

如果您启用了 Chrome 同步功能，以下数据会在您登录的 Chrome 设备间同步：

  • 显示设置和偏好

重要：订阅链接、API 密钥等敏感信息仅存储在本地，不会参与云同步，确保您的隐私安全。

2. 数据的使用

我们使用收集的信息用于：

  • 连接和管理 Clash：与您的本地 Clash API 通信，管理代理节点
  • 显示流量信息：向订阅服务器发送 HEAD 请求获取流量使用情况
  • 保存设置：记住您的偏好设置，提供更好的使用体验
  • 节点测试：测试节点延迟，帮助您选择最快的节点

3. 数据的共享

我们不会与任何第三方共享您的数据。

您的数据仅用于以下情况：

  • 与您本地运行的 Clash API 通信（127.0.0.1 或 localhost）
  • 向您的订阅服务商的服务器发送请求（仅用于获取流量信息）
  • Chrome 浏览器的同步功能（仅在您启用时，且仅同步非敏感设置）

4. 数据安全

我们采取以下措施保护您的数据：

  • 本地存储：所有敏感数据（订阅链接、API 密钥）仅存储在您的本地设备
  • 敏感数据遮蔽：在界面上默认隐藏订阅链接中的敏感参数（如 token、key）
  • HTTPS 加密：所有外部请求都使用 HTTPS 加密传输
  • 数据导出：提供配置导出功能，您随时可以备份自己的数据
  • 数据清除：提供"彻底重置"功能，您可以随时清除所有存储的数据

5. 您的权利

您对自己的数据拥有完全的控制权：

  • 访问权：您可以通过设置页面查看所有存储的数据
  • 修改权：您可以随时修改、添加或删除订阅链接和设置
  • 删除权：您可以通过"彻底重置"功能清除所有数据
  • 导出权：您可以将配置导出为 JSON 文件进行备份
  • 导入权：您可以从备份文件恢复配置

6. 第三方服务

ProxyHub 可能与以下第三方服务交互：

  • Clash API：运行在您本地设备（127.0.0.1）的代理服务
  • 订阅服务商：您配置的订阅链接对应的服务提供商

这些第三方服务有自己的隐私政策。我们建议您查阅这些服务的隐私政策。

7. Cookie 和跟踪技术

ProxyHub 不使用任何 Cookie 或跟踪技术。

8. 儿童隐私

ProxyHub 不面向 13 岁以下儿童。我们不会故意收集 13 岁以下儿童的个人信息。

9. 政策变更

我们可能会不时更新本隐私政策。变更后的政策将在本页面发布，并在政策顶部标明最后更新日期。重大变更时，我们会通过扩展更新通知您。

11. 法律依据

本隐私政策符合以下法律法规的要求：

  • 欧盟通用数据保护条例 (GDPR)
  • 加州消费者隐私法案 (CCPA)
  • Chrome Web Store 隐私政策要求
