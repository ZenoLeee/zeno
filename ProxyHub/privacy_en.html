ProxyHub Privacy Policy

Last Updated: February 3, 2025

We are committed to protecting your privacy. All data for this extension is stored locally on your device. We do not collect, store, or transmit any personal data to our servers.

1. Information We Collect

1.1 Locally Stored Data

ProxyHub uses the Chrome Storage API to store the following information on your local device:

  • Subscription URLs: Clash subscription URLs you add (including any authentication tokens)
  • API Configuration: Clash API address and secret key
  • Display Settings: Your interface preferences (such as dark mode, dual-column display, etc.)
  • Node Favorites: Your list of favorited nodes
  • Traffic Information: Traffic usage retrieved from subscription URLs (stored locally only)

1.2 Cloud Sync Data (Optional)

If you enable Chrome sync functionality, the following data will be synchronized across your Chrome-signed-in devices:

  • Display settings and preferences

Important: Sensitive information such as subscription URLs and API keys is stored locally only and will not participate in cloud sync to ensure your privacy and security.

2. How We Use Your Data

We use the collected information for:

  • Connect and Manage Clash: Communicate with your local Clash API to manage proxy nodes
  • Display Traffic Information: Send HEAD requests to subscription servers to retrieve traffic usage
  • Save Settings: Remember your preferences to provide a better user experience
  • Node Testing: Test node latency to help you choose the fastest node

3. Data Sharing

We do not share your data with any third parties.

Your data is only used for:

  • Communication with your local Clash API (127.0.0.1 or localhost)
  • Sending requests to your subscription service provider's servers (for traffic information only)
  • Chrome browser sync functionality (only when enabled, and only syncs non-sensitive settings)

4. Data Security

We take the following measures to protect your data:

  • Local Storage: All sensitive data (subscription URLs, API keys) is stored only on your local device
  • Sensitive Data Masking: Sensitive parameters (such as tokens, keys) in subscription URLs are hidden by default in the interface
  • HTTPS Encryption: All external requests use HTTPS encrypted transmission
  • Data Export: Configuration export functionality is provided so you can backup your data at any time
  • Data Clearing: A "Reset All" feature is provided so you can clear all stored data at any time

5. Your Rights

You have complete control over your data:

  • Right to Access: You can view all stored data through the settings page
  • Right to Modify: You can modify, add, or delete subscription URLs and settings at any time
  • Right to Delete: You can clear all data through the "Reset All" feature
  • Right to Export: You can export your configuration as a JSON file for backup
  • Right to Import: You can restore your configuration from a backup file

6. Third-Party Services

ProxyHub may interact with the following third-party services:

  • Clash API: Proxy service running on your local device (127.0.0.1)
  • Subscription Providers: Service providers corresponding to the subscription URLs you configure

These third-party services have their own privacy policies. We encourage you to review the privacy policies of these services.

7. Cookies and Tracking Technologies

ProxyHub does not use any cookies or tracking technologies.

8. Children's Privacy

ProxyHub is not directed to children under the age of 13. We do not knowingly collect personal information from children under 13.

9. Policy Changes
